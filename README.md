# User Management System

## Tech Stack
- Java Servlets
- JSP
- JDBC
- PostgreSQL

## Features
- User Registration
- Input Validation Layer
- DAO Pattern (JDBC)
- MVC Architecture
- Error Handling

## Project Flow
User → Servlet → Validator → DAO → Database → JSP

## Setup Instructions
1. Clone repository
2. Configure PostgreSQL DB
3. Update DB credentials
4. Run on Apache Tomcat
